Python is a general-purpose, high-level programming language originally designed by Guido van Rossum in 1991. It emphasizes code readability and concepts in fewer lines of code.

1.1 Introduction to Python

History & Versions

Python 2 (2000) vs Python 3 (2008). Python 3 includes new features and better testing.

How it Works

Code is translated into byte code (.pyc) and executed by the PVM (Python Virtual Machine).

Modes of Operation

Interactive Mode

Execute code directly at the prompt >>>. Best for testing small snippets. Memory quantities not shown unless requested.

Script Mode

Source code stored in a file .py. Executed via python MyFile.py. Essential for longer programs.

1.2 Data Types

Standard types define operations and storage methods.

Number

Int

Whole numbers without decimals.

20, -5, 0b10
Number

Float

Decimals or scientific notation.

2.8, -87.7e100
Logic

Boolean

Represents truth values.

True, False
Text

String

Immutable set of characters in ' or ". Escape with \ (e.g., \n for new line).

"Hello", 'World'
Collection

List

Ordered, changeable, allows duplicates.

[1, "a", 3.5]

1.3 Variables, Expressions & Statements

Variable Rules

  • Start with a letter or underscore.
  • Cannot start with a number.
  • Case-sensitive & alphanumeric only.
# Assignment Examples
x = 10 # Standard
a = b = c = 1 # Multiple Assign (Single Value)
a, b = 1, 2 # Multiple Object Assign
# Conditional Expression
x = "1" if True else "2"

1.4 Operators & Comments

Precedence

Determines evaluation order. * is higher than +.

Tip: Use parentheses () to override precedence.

1.5 Modules & Functions

Modules

Files containing definitions (.py). Used via import.

e.g., import sys

Functions

Reusable code blocks performing specific tasks.

Built-in: print() | User-defined: def name():

Argument Type Description Syntax Example
Default Value assigned in definition. def func(a=4)
Keyword Passed by name, order doesn't matter. func(c=50, a=100)
Variable-length Unknown number of arguments. *args (tuple), **kwargs (dict)
