4.2 Tuples

üìï

Immutable Sequence

Ordered, Unchangeable. Uses round brackets ().

  • Faster than lists.
  • Can be used as dictionary keys.
  • Tuple Assignment: a, b = 1, 2

Watch Out!

(i for i in 'abc') is NOT a tuple comprehension. It creates a Generator Object.

4.3 Dictionaries

Unordered collection of Key-Value pairs. Indexed by keys, not numbers. Uses curly brackets {}.

my_dict = {"brand": "mrcet", "year": 2004}
KEY VALUE
"brand" "mrcet"
"year" 2004

Dictionary Methods

  • keys() / values() / items()
    Returns views of keys, values, or pairs.
  • get(key)
    Safe access. Returns None if key missing.
  • pop(key)
    Removes item and returns value.
